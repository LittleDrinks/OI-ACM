# 二分图带权最大匹配
KM 算法 = 匈牙利算法 + 扩大相等子图
每个点有一个顶标。如果一条边 $(i,j)$ 满足 $lx[i] + ly[j] = w(i,j)$，那么称其为相等边，相等边构成的子图称为相等子图。
