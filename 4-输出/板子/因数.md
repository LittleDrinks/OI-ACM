可以在 $O(\sqrt{V})$ 的时间内求出一个数的因数。
$[1,n]$ 所有数字的因数个数和是 $O(n\log n)$ 的。
$720720$ 是 $10^6$ 内因数最多的数字，其因数个数为 $240$。
$735134400$ 是 $10^9$ 内因数最多的数字，其因数个数为 $1344$。
$10^{12}$ 以内的因数大约是 $3\times 10^3$ 数量级。
```cpp
const int X=1e5;
vector<int> fac[X+5];
for (int i = 1; i <= X; ++i) {
    for (int j = i; j <= X; j += i) {
        fac[j].push_back(i);
    }
}
```
