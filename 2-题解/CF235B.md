## [CF235B. Let's Play Osu!](https://codeforces.com/problemset/problem/235/B)

设 $X$ 表示连击次数。则有
$$\mathbb{E}(X+1)^2=\mathbb{E}(X^2)+2\mathbb{E}(X)+1$$
我们发现如果连击次数增加一，答案的增量是 $2\mathbb{E}(X)+1$
设 $f(i)$ 表示前 $i$ 次操作结束后当前连击的期望。$f(i-1)$ 有 $1-p$ 的概率清零，有 $p$ 的概率加一，即：
$$
f(i) = p\times\bigg(f(i-1) + 1\bigg)
$$
$g(i)$ 表示期望得分，首先它应该无脑继承 $g(i-1)$，再以 $p$ 的概率增加一个 $2\mathbb{E}(X)+1$，则
$$
g(i)=g(i-1) + p\times \bigg( 2f(i-1) + 1\bigg)
$$
时间复杂度 $O(n)$

[***AC 代码***](https://codeforces.com/contest/235/submission/351416151)

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
using db = long double;
using i128 = __int128;

void solve()
{
    int n;
    cin >> n;
    db len = 0, ans = 0;
    for (int i = 1; i <= n; ++i) {
        db p;
        cin >> p;
        ans += p * (2 * len + 1);
        len = p * (len + 1);
    }
    cout << fixed << setprecision(15) << ans << "\n";
}

int main()
{
    ios::sync_with_stdio(0); cin.tie(0); cout.tie(0); 
    int t = 1;
    // cin >> t;
    while (t--) { solve(); }
}
```
